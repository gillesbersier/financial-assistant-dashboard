# Financial Assistant Dashboard

A premium, intelligent financial dashboard designed for instant clarity and control.

## Tech Stack
- **Framework**: Next.js 16 (App Router)
- **Styling**: Vanilla CSS Modules + CSS Variables (Design System)
- **Components**: Custom built with React
- **Charts**: Recharts
- **Icons**: Lucide React
- **State**: Zustand (Ready for integration) -> *Currently using Mock Data*

## Features
- **Smart Statistics**: Key financial metrics with trend analysis.
- **Monthly Spending**: Interactive bar chart with spending history.
- **Automation Status**: Real-time visibility into N8N/AI workflows.
- **Recent Invoices**: Status-coded transaction list with filters.
- **Responsive Design**: Mobile-friendly layout.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000)

## Design System
- **Fonts**: Inter (Google Fonts)
- **Palette**: Neutral Slate with Indigo Accents
- **Dark Mode**: System preference supported (automatic)
